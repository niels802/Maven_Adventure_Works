## Key Topics Covered

This project is based on the **Microsoft Power BI Desktop course by Maven Analytics** and covers the full end-to-end analytics workflow.

Throughout this project, the following key topics are demonstrated:

### Introducing Power BI Desktop  
Understanding the Power BI environment, interface, and core components used to build data reports and dashboards.

### Connecting & Shaping Data  
Importing data from various sources and transforming it using Power Query to ensure it is clean, structured, and analysis-ready.

### Creating a Data Model  
Building relationships between tables and designing a scalable data model to support efficient reporting and analysis.

### Adding Calculated Fields with DAX  
Developing calculated columns and measures using DAX to create advanced metrics and business logic.

### Visualizing Data with Reports  
Designing interactive and user-friendly reports that communicate insights effectively.

### AI Visuals in Power BI  
Exploring built-in AI visuals to uncover patterns, trends, and key drivers within the data.

###  Optimization Tools  
Applying performance optimization techniques to improve report efficiency and usability.
